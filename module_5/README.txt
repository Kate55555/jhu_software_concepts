Module 3 - Databases

Approach: This project uses Python 3.13.2. The core .py files inculude:

create_table.py - I used it to create the database and table structure.

load_data.py - This file loads the data from applicant_data.json to the database. To load another portion of data from pplicant_data.json, download the module_3 files, open a terminal window, and type "python load_data.py".

query_data.py - This file queries the database to answer the Module 3 assignment questions. To get the answers I get out of the GradCafe entries, download the module_3 files, open a terminal window, and type "python query_data.py".

app.py - Orchestrates the Flask application. To run the Flask application, download the module_3 files, open a terminal window, and type "python app.py".

Known Bugs: None.
